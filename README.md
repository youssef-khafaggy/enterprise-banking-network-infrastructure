# enterprise-banking-network-infrastructure
Enterprise banking network infrastructure built using Cisco, FortiGate, and Windows Server technologies. Features VLAN segmentation, Inter-VLAN Routing, VTPv3, VRRP, EtherChannel (PAgP), OSPF, Active Directory, DHCP, SD-WAN, Web Filtering, and Port Security to provide a secure, scalable, and highly available enterprise network environment

# Technologies Used

- Cisco IOS
- VLANs
- VTP Version 3
- Inter-VLAN Routing
- VRRP
- EtherChannel (PAgP)
- OSPF
- FortiGate Firewall
- SD-WAN
- Active Directory (AD DS)
- DHCP
- Windows Server
- Web Filtering
- Port Security
- GNS3

---

## Project Objective

To design and implement a secure, scalable, and highly available banking network infrastructure that integrates switching, routing, security, SD-WAN, and Windows Server services while following enterprise networking best practices.
