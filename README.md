# enterprise-banking-network-infrastructure
Enterprise banking network infrastructure built using Cisco, FortiGate, and Windows Server technologies. Features VLAN segmentation, Inter-VLAN Routing, VTPv3, VRRP, EtherChannel (PAgP), OSPF, Active Directory, DHCP, SD-WAN, Web Filtering, and Port Security to provide a secure, scalable, and highly available enterprise network environment
